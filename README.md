## Creating My AWS Account

In this step, I'm going to sign up for a new AWS account, verify my email address, and create a strong root user password.

### Understanding the root user

I learned that a root user is the original, all-powerful identity created when you first sign up for an Amazon Web Services account. A strong password is important because the root account holds the "keys to the kingdom." If an unauthorized person compromises a root password, they gain total control over the system to steal data, install malware, or lock out the rightful owner.

## Setting Up Billing and Payment

In this step, I'm going to set up the billing information. I'll choose a free account plan, fill up the contact information needed, then add a payment method.

### Why AWS requires a payment method

AWS requires a payment method during sign-up to verify your identity, prevent fraud and bot abuse, and ensure a seamless billing transition if you exceed your promotional credits or Free Tier limits.

I verified my identity by providing my phone number and entering the verification code sent by AWS during the account setup process.
